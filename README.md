# Ngineer ⚙️

**Ngineer** solves the headache of writing complex Nginx proxy, routing, and WebSocket configs by making it visual and real-time. No more syntax errors or forgotten headers.

🔗 **[Live Demo](https://wael.work.gd/ngineer)**

## ✨ Features

- **Zero-Install Execution:** No Node.js, no Webpack, no build step. It runs entirely in the browser.
- **Vue.js Reactivity:** Centralized state management ensures the configuration template updates instantly as you toggle settings.
- **Retro-Modern Glassmorphic UI:** A beautifully crafted, responsive interface featuring dynamic themes (including a custom Wael Mode) and butter-smooth transitions.
- **Advanced Routing:** One-click toggles for Reverse Proxies, WebSockets, SPA Fallbacks (`try_files`), and strict security headers.

## 🚀 Quick Start / Local Dev

Because Ngineer is a zero-install static application, running it locally is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Wael/Ngineer.git
   cd Ngineer
   ```
2. **Open it:**
   Double-click `index.html` to open it in any modern web browser.

That's it! You're ready to generate perfect Nginx configurations.
